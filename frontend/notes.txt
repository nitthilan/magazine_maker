Select2 notes:
	- possible options: 
		- { maximumSelectionSize: 3 } - max number of multiple selections
		- minimumInputLength: 2 - num characters for search to work
		- allowClear: true - enables clearing all selections
		- placeholder: "Select a State" - place holder text
		- tags:["red", "green", "blue"] - enables tags in select2
		- type anything to add into the tag
		- maximumInputLength: 10 - maximum words allowed for a tag
		- tokenSeparators: [",", " "] - add tags as and when the token separator appears
		- minimumResultsForSearch - number of results to be shown for search
		- closeOnSelect - default true, if false, keeps the search open for multiple selection
